## 📖 فهرست مطالب

### **فصل اول:** تاجر و اعمال تجاری
- تعریف تاجر (ماده ۱ ق.ت)
- اعمال تجاری ذاتی (۱۰ مورد)
- اعمال تجاری تبعی (۴ مورد)
- اصل معاملات تاجر
- ثبت تجارتی = اماره (نه دلیل قطعی)
- غیرمنقول = هرگز تجاری
- برات = همیشه ذاتی
- تاجر نیستند: قائم مقام، مدیران، قیم صغیر

### **فصل دوم:** دفاتر تجاری
- ۴ دفتر تجاری (روزنامه، کل، دارایی، کپیه)
- امضاهای قانونی
- تعارض دفاتر = هر دو ساقط
- موافقت دفاتر = دفتر فروشنده
- اعتبار دفاتر

### **فصل سوم:** دلال
- تعریف و وظایف (مواد ۳۳۵–۳۵۵)
- منتفع و سهیم
- زمان استحقاق اجرت
- مخارج = اصلاً قابل مطالبه نیست
- حق حبس = ندارد
- طرف معامله = نمی‌تواند

### **فصل چهارم:** حق‌العمل‌کار
- روابط سه‌گانه (ماده ۳۵۸)
- قیمت فروش (بیشتر/کمتر)
- زمان استحقاق = اجرا
- مخارج = قابل مطالبه است
- حق حبس = دارد
- مظنه بورسی

### **فصل پنجم:** متصدی حمل‌ونقل
- تعهد به نتیجه
- عدل‌بندی (ماده ۳۸۶)
- مسئولیت تضامنی
- مهلت دعوا (۱ سال / ۸ روز)
- حق حبس

### **فصل ششم:** قائم مقام تجارتی و نمایندگان
- تعریف و انتخاب
- **مهم:** فوت/حجر رئیس = منعزل نمی‌شود
- اصل استقلال (نه اجتماع)

### **فصل هفتم:** ضمانت
- اصل: نقل ذمه
- ۴ استثناء ضم ذمه (اِش مِد)
- تضامنی و وثیقه‌ای
- احکام خاص
